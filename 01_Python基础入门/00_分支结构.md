# 任何程序都由这3种基本结构组合而成。

    # 顺序结构是程序按照线性顺序依次执行的一种运行方式。
    # 语句块1 -->  语句块2
    
    # 分支结构是程序根据条件判断结果而选择不同向前执行路径的一种运行方式
        # 单分支控制
        # 语法格式：
                    if  <条件>:   # 满足条件执行下面的语句，不满足则跳过。
                        <语句块>

                            例如：
                            # 通过eval函数确认用户输入的值是否是数字，如果是字符串则报错。
                            n = eval(input("请输入分数："))
                            
                            if n >= 60:
                                print("成绩合格，考试通过")     
        # 二分支结构
                    if  <条件>:   # 满足条件执行下面的语句1，不满足则跳过，执行下面的语句2。
                        <语句块1>
                    else:
                        <语句块2>

        # 多分支结构
        多分支结构一般用于判断同一个条件或一类条件的多个执行路径。
        依次判断条件并执行第一个条件为True的语句。若没有条件成立，执行else下面的语句。

                    if  <条件>:   # 满足条件执行下面的语句1，不满足则跳过，执行下面的语句2。

                    elif <条件2>:
                        <语句块2>
